# 🎓 EnglishPro PWA

Додаток для вивчення англійської мови з AI-репетитором.

---

## 🚀 Як запустити (3 простих кроки)

### Крок 1: Встановити Node.js
Завантажте з https://nodejs.org (версія 18+)

### Крок 2: Встановити залежності та запустити
Відкрийте термінал (або Command Prompt) у папці проєкту та виконайте:

```bash
npm install
npm run dev
```

Відкрийте в браузері: **http://localhost:5173**

### Крок 3: Зібрати для продакшену
```bash
npm run build
```
Готові файли будуть у папці `dist/`

---

## 🌐 Як опублікувати безкоштовно (Vercel)

1. Створіть акаунт на https://vercel.com (через GitHub)
2. Залийте цей проєкт на GitHub
3. На Vercel натисніть "New Project" → оберіть репозиторій
4. Натисніть "Deploy" — готово!

Vercel дасть вам URL типу `englishpro.vercel.app`

### Альтернатива: Netlify
1. https://netlify.com → "Add new site" → "Deploy manually"
2. Перетягніть папку `dist/` на сайт
3. Готово!

---

## 📱 Як встановити на телефон

Після публікації, відкрийте URL на телефоні:

**iPhone (Safari):**
1. Відкрийте сайт у Safari
2. Натисніть кнопку "Поділитися" (квадрат зі стрілкою)
3. Прокрутіть вниз → "На Початковий екран"
4. Натисніть "Додати"

**Android (Chrome):**
1. Відкрийте сайт у Chrome
2. Натисніть ⋮ (три крапки зверху)
3. "Додати на головний екран"
4. Натисніть "Додати"

Додаток з'явиться як іконка на вашому телефоні!

## 🤖 AI-функції (Claude API)

Для AI-чату, перевірки тексту та генерації слів потрібен API-ключ:

1. Зареєструйтесь на https://console.anthropic.com
2. Створіть API-ключ
3. В додатку: ⚙️ Налаштування → вставте ключ

**Без ключа** додаток працює повністю — словник, граматика, 
картки, ігри, історії, аудіювання. AI-функції просто будуть недоступні.
